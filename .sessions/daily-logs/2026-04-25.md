#### Batch 2 — Core Kafka Services (Tasks 4–6)

- **`src/common/kafka/schema-registry.service.ts`** — wraps `@kafkajs/confluent-schema-registry`:
  - `registerAll()`: reads all `.avsc` files, sets BACKWARD compat via HTTP PUT, registers schemas, caches `subject → schemaId`; idempotent
  - `encode<T>(subject, payload)`: uses cached schema ID; throws if schema not registered
  - `decode<T>(buffer)`: auto-resolves schema from magic byte in Avro buffer
- **`src/common/kafka/kafka-producer.service.ts`** — raw KafkaJS producer:
  - `send<T>(topic, payload)`: calls `schemaRegistry.encode(topic+'-value', payload)`, then `producer.send()`
  - `onModuleInit()`/`onModuleDestroy()`: connect/disconnect
- **`src/common/kafka/kafka-consumer.service.ts`** — raw KafkaJS consumer:
  - `subscribe<T>(topic, handler)`: registers handler in Map; call before `startConsuming()`
  - `startConsuming()`: connects, subscribes to all registered topics, runs consumer loop with Avro decode
  - `onModuleDestroy()`: disconnect
- **`src/common/kafka/kafka.module.ts`** — `@Global()` module exporting all 3 services

#### Batch 3 — AppModule + Producer Migration (Tasks 7–9)

- `AppModule` imports `KafkaModule`, implements `OnApplicationBootstrap`:
  - `onApplicationBootstrap()`: `await registerAll()` → `await startConsuming()` (guarantees schema registration before consumer starts)
- All 3 producers migrated from `ClientKafka.emit()` to `KafkaProducerService.send()`:
