### Key Decisions

1. **MinIO `forcePathStyle: true`**: Required for localhost/Docker MinIO — AWS SDK defaults to virtual-hosted style which fails without a real domain
2. **MinIO dual registration**: `MinioService` registered in both `AppModule` (global) and `ProductModule` (local) — NestJS module scoping requires it where injected
3. **`@types/multer` missing**: Not a transitive type — must be installed explicitly when using `Express.Multer.File`
4. **PostgreSQL COUNT → string**: Raw SQL `COUNT(*)` returns strings — explicit `Number()` cast required in `complianceReport()` return mapping
5. **Intra-module JOIN is safe**: `ExportDocumentPdfService` JOINs `ExportDocument` + `Certification` — both in `certification` schema, not cross-module. Direct repository usage is architecturally correct.
6. **YAGNI Lab enforcement**: US-030 delivers metadata registry only. No enforcement on `POST /lab-tests` in v1 — `labId` FK deferred to Phase 2.
7. **Route ordering**: `GET /compliance-report` and `GET /onssa-report` registered before `GET /:id` — confirmed platform pattern (literal segments before param segments)
