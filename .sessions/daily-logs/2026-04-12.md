### Decisions

1. **PDFKit Arabic shaping**: Amiri-Regular.ttf with `align: 'right'` — full ligature shaping via `pdfkit-arabic` is Phase 2 enhancement
2. **jest.mock('fs') anti-pattern**: Blanket fs mock breaks TypeORM's `path-scurry` module. Use spies or rely on real fs with non-existent paths in test environments
3. **CACHE_MANAGER in service constructors**: Any service that injects `CACHE_MANAGER` requires a mock in ALL test modules that use that service, even if the test doesn't exercise caching
4. **TM-3 pattern**: This is the 2nd sprint TM-3 has been deferred — consider dedicating a separate infra session to spin up Docker and run migrations
5. **Backlog summary table**: Should be updated after every sprint as Done counts drift from individual rows
