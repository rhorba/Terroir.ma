### Decisions

| Decision | Rationale |
|----------|-----------|
| `as unknown as Response` cast on `intlMiddleware(req)` return | `NextResponse` extends `Response` at runtime; type divergence is next-auth v5 beta artifact |
| `jose` warnings accepted as benign | JWE deflate not triggered; `auth.config.ts` split deferred to FE-S9 |
| Server action for Keycloak sign-in | No client JS needed for auth redirect; cleaner Server Component |
| Layout-level `auth()` guard in each role shell | Belt-and-suspenders; handles edge case where session cookie drops between middleware and render |
| Sidebar nav hardcoded to `/fr/` prefix | next-intl `useRouter` migration is FE-S9 polish scope |
